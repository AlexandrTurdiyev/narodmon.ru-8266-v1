# narodmon.ru-8266-v1 #
___

## Спецификация ##
- Отладочная плата NodeMCU на базе ESP8266.
- Датчик температуры и влажнсти DHT22.
- Датчик температуры DS18B20.
- Датчик атмосферного давления BMP180.
